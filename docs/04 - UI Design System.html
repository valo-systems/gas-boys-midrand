Google Reviews
Places API → Laravel cache → Custom React widget
How It Works — Architecture
1
Google Places API
Laravel cron runs every 6 hrs → fetches latest reviews via places/details → stores in MySQL google_reviews table.
2
REST Endpoint
GET /api/reviews returns cached reviews — no API key exposed client-side, instant load, no third-party iframe.
3
Custom React Widget
Branded card grid matching Gas Boys design system. Stars, reviewer name, date, Google icon. No generic embed styling.

Laravel Schema — google_reviews table
Schema::create('google_reviews', function (Blueprint $table) {
  $table->id();
  $table->string('author_name');
  $table->string('profile_photo_url')->nullable();
  $table->tinyInteger('rating');
  $table->text('text');
  $table->string('relative_time_description');
  $table->timestamps();
});
